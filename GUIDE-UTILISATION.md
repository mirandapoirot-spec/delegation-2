# 📖 Guide d'Utilisation - Nouvelles Fonctionnalités

## 🎯 Vue d'Ensemble

Ce guide vous explique comment utiliser les 4 nouvelles fonctionnalités majeures ajoutées au portail MINJEC.

---

## 1. 💬 Messagerie Interne

### Accès
- **URL:** `messaging.html`
- **Menu:** Plus > Messagerie
- **Bouton flottant:** Icône de chat en bas à droite

### Comment utiliser

#### Pour les Clients:
1. Connectez-vous à votre compte
2. Accédez à la messagerie
3. Vous verrez automatiquement une conversation avec l'administrateur
4. Cliquez sur la conversation pour l'ouvrir
5. Tapez votre message dans le champ en bas
6. Appuyez sur Entrée ou cliquez sur l'icône d'envoi

#### Pour les Admins:
1. Vous voyez toutes les conversations avec les clients
2. Cliquez sur une conversation pour répondre
3. Les nouveaux messages sont marqués avec un badge

### Fonctionnalités:
- ✅ Messages en temps réel
- ✅ Recherche de conversations
- ✅ Statut en ligne
- ✅ Historique complet
- ✅ Pièces jointes (à venir)

---

## 2. 📅 Système de Rendez-vous

### Accès
- **URL:** `appointments.html`
- **Menu:** Plus > Rendez-vous
- **Bouton flottant:** Icône de calendrier en bas à droite

### Comment prendre un rendez-vous

1. **Sélectionner une date:**
   - Cliquez sur un jour dans le calendrier
   - Les jours avec des rendez-vous ont un point bleu

2. **Choisir un créneau:**
   - Les créneaux disponibles s'affichent
   - Les créneaux réservés sont grisés
   - Cliquez sur un créneau disponible

3. **Remplir le formulaire:**
   - Type de consultation (orientation, projet, financement, etc.)
   - Motif (optionnel mais recommandé)
   - Priorité (basse, moyenne, haute)

4. **Confirmer:**
   - Vérifiez le récapitulatif
   - Cliquez sur "Confirmer"

### Gérer vos rendez-vous:
- Voir tous vos rendez-vous dans la colonne de droite
- Annuler un rendez-vous (bouton rouge)
- Modifier un rendez-vous (bouton bleu)

### Statuts:
- 🔵 **En attente:** Rendez-vous créé, en attente de confirmation admin
- 🟢 **Confirmé:** Rendez-vous validé par l'admin
- 🔴 **Annulé:** Rendez-vous annulé

## 3. 📊 Analytiques

### Accès
- **URL:** `analytics.html`
- **Menu:** Plus > Analytiques
- **Recommandé pour:** Administrateurs

### Que voir sur cette page

#### Statistiques Principales:
- **Utilisateurs actifs:** Nombre total d'utilisateurs
- **Projets soumis:** Tous les projets
- **Projets validés:** Projets approuvés
- **Jeunes formés:** Nombre de bénéficiaires

#### Graphiques:

1. **Projets par mois (Barres):**
   - Évolution mensuelle des soumissions
   - Filtres: 6 mois / 1 an

2. **Statuts des projets (Camembert):**
   - Répartition: En attente / Validés / Refusés
   - Pourcentages en temps réel

3. **Évolution des inscriptions (Ligne):**
   - Tendance des nouvelles inscriptions
   - Filtres: Semaine / Mois / Année

4. **Top Projets par Région (Tableau):**
   - Classement des régions
   - Taux de réussite
   - Barres de progression

### Export des données:
- Cliquez sur "Exporter" en haut à droite
- Télécharge un fichier JSON avec toutes les données
- Utilisable pour des analyses externes

## 4. 🎫 Centre de Support

### Accès
- **URL:** `support.html`
- **Menu:** Plus > Support
- **Bouton flottant:** Icône de casque en bas à droite

### Créer un ticket

1. **Cliquez sur "Nouveau Ticket"** dans la sidebar

2. **Remplissez le formulaire:**
   - **Catégorie:** 
     - Problème technique
     - Gestion de compte
     - Question sur un projet
     - Financement
     - Autre
   
   - **Sujet:** Titre court et clair
   
   - **Description:** Expliquez votre problème en détail
   
   - **Priorité:**
     - 🟢 Basse: Question générale
     - 🟡 Moyenne: Problème gênant
     - 🔴 Haute: Problème bloquant

3. **Créer le ticket**

### Suivre vos tickets:

- **Tous:** Voir tous vos tickets
- **Ouverts:** Tickets en attente de réponse
- **En cours:** Tickets traités par l'équipe
- **Résolus:** Tickets fermés

### Consulter la FAQ:

1. Cliquez sur "FAQ" dans la sidebar
2. Parcourez les questions fréquentes
3. Cliquez sur une question pour voir la réponse

### Statistiques:
- Voir le nombre de tickets par statut
- Temps de réponse moyen
- Historique complet

## 🔧 Dépannage

### Problèmes Courants:

**1. Les messages ne s'affichent pas:**
- Rafraîchissez la page (F5)
- Vérifiez votre connexion
- Videz le cache du navigateur

**2. Le calendrier ne charge pas:**
- Vérifiez que JavaScript est activé
- Essayez un autre navigateur
- Contactez le support

**3. Les graphiques ne s'affichent pas:**
- Attendez quelques secondes
- Vérifiez votre connexion internet
- Rechargez la page

**4. Impossible de créer un ticket:**
- Vérifiez que tous les champs sont remplis
- Essayez de vous reconnecter
- Contactez directement l'admin
